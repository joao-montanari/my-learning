**Conteúdo:** Conheça palavras-chave fáceis e úteis para descrever os seus sentimentos no dia a dia, como alegria, tristeza e cansaço.
**Especificações:** Vocabulary - 03 Atividades
**Professor(a):** Tiago Rocha

### Vocabulário

| **FRASE** | **TRADUÇÃO**  |
| --------- | ------------- |
| I´m sad   | Estou triste  |
| I´m tired | Estou cansado |
| I´m ok    | Tudo certo    |
| I´m fine  | Estou bem     |
| I´m good  | Estou bem     |
| I´m Happy | Estou feliz   |

### Exercícios
**Questão 01:** Você acabou de receber uma ótima notícia e quer contar como está se sentindo. Qual frase você usa?
- **A)** I´m sad
- **B)** I´m tired
- **C)** I´m happy
- **D)** I´m OK

**Questão 02:** Um colega te pergunta como você está e você quer dar uma resposta rápida e neutra para dizer que está bem. Qual alternativa tem o mesmo sentido de “I’m fine”?
- **A)** I´m sad
- **B)** I´m tired
- **C)** I´m good
- **D)** I´m happy
