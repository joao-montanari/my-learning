
**Conteúdo:** Vamos observar uma conversa entre amigos reagindo e descrevendo como se sentem para você aprender a "pescar" os sentimentos no meio do diálogo.
**Especificações:** Comprehension 03 Atividades
**Professor(a):** David Medeiros

### Vocabulário
| **FRASE**                    | **TRADUÇÃO**                      |
| ---------------------------- | --------------------------------- |
| How are you?                 | Como você está?                   |
| I´m Happy                    | Estou feliz                       |
| I´m happy for you            | Estou feliz por vocês             |
| I´m fine                     | Estou bem                         |
| Really?                      | Sério?                            |
| Absolutely, I´m totally fine | Com certeza, estou totalmente bem |

### Exercícios
**Questão 01:** Leia a frase e escolha a alternativa que tem o mesmo significado do que ele disse: "I´m good".
- **A)** I´m sad
- **B)** I´m fine
- **C)** I´m happy
- **D)** I´m tired

**Questão 02:** Leia a frase e selecione a alternativa correta sobre o que ele quis dizer: "Absolutely, I´m totally fine".
- **A)** I´m sad
- **B)** I´m tired
- **C)** I´m fine
- **D)** I´m happy

**Questão 03:** Sua amiga conseguiu o emprego dos sonhos e você quer comemorar dizendo "Estou feliz por você". Como você diz isso em inglês?
- **A)** I´m happy
- **B)** I´m so happy
- **C)** I´m happy for you
- **D)** I´m so happy

**Questão 03:** Um amigo diz que está bem, mas pelo tom de voz você desconfia. O que você diz para questionar a resposta dele?
- **A)** I´m happy
- **B)** I´m tired
- **C)** really?
- **D)** I´m fine
